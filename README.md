## Code in Action

```
# import the math module
fetch std_math;

obj x = 0;

# let's go for a walk!
walk i = 0 through 10 {
    obj x = x + 1;

    if x == 5 {
        leave;
    }
}

# print the value of 'x'
bark(x);

# greet someone
func greet(name) {
    bark("Hello, " + name + "!");
}

greet("John Doe");

bark("Pi is equal to: " + tostring(math_pi));
bark("We have reached the end of our program. I hope you enjoyed!");
```
